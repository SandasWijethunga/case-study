Impact of Demographic Factors on Young Adults' Music Listening Habits

This project is a comprehensive data science case study examining how demographic characteristics (age, gender, degree program) and genre preferences influence the daily music consumption of young adults. The study utilizes primary data collected via field surveys and rigorous statistical analysis performed in R.
